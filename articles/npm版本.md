## 一、安装npm插件n ,通过n模块来管理node版本

### 1、全局安装n模块

```
npm instlal -g n
```

### 2、安装当前稳定版本

```
n stable或者sudo n stable
```

### 3、安装最新版本的

```
n latest或者sudo n latest
```

### 4、安装指定版本的node

```
n v8.16.0
```

### 5、卸载指定的node版本

```
n rm v8.16.0
```
